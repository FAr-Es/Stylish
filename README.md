# Stylish 👕🛍️

A Flutter application for a **Clothes Store** project.\
This app is still under development 🚧.

------------------------------------------------------------------------

## ✨ Features 

-   User Signup & Login
-   Browse clothes categories
-   Add items to cart
-   Checkout system
-   Modern & clean UI
