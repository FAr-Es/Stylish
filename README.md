## 🚀 Getting Started

1.  Clone the repository:

    ``` bash
    git clone https://github.com/FAr-Es/Stylish.git
    ```

2.  Navigate to the project folder:

    ``` bash
    cd Stylish
    ```

3.  Get the dependencies:

    ``` bash
    flutter pub get
    ```

4.  Run the app:

    ``` bash
    flutter run
    ```

------------------------------------------------------------------------

## 📂 Project Structure

    lib/
      pages/        -> app pages (login, signup, etc.)
      widgets/      -> reusable custom widgets
      constant/     -> colors, constants, etc.
      main.dart     -> entry point
